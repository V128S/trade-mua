Прошивка — це операційна система вашого ASIC-майнера. Від неї залежить, скільки хешрейту видає машина, як вона реагує на перегрів і наскільки стабільно працює цілодобово. Розберемо, навіщо взагалі оновлювати або змінювати прошивку і як це зробити без ризику для обладнання.

## Навіщо прошивати майнер?

Виробники регулярно випускають оновлення, які:

- **Виправляють баги** — нестабільна робота хешбордів, помилки в логах, некоректне відображення хешрейту.
- **Підвищують стабільність** — кращий тепловий контроль, захист від перегріву.
- **Додають нові функції** — підтримка нових пулів, поліпшений інтерфейс, автоналаштування.
- **Підвищують ефективність** — деякі оновлення дають +1–3% хешрейту без зміни енергоспоживання.

Крім офіційної прошивки, існують **кастомні прошивки** (наприклад, Vnish, BraiinsOS+), які відкривають додаткові можливості: авто-тюнінг під напругу, детальна статистика по чипах, режими Eco/Turbo.

## Офіційна vs. кастомна прошивка

| Параметр | Офіційна | Кастомна (Vnish, BraiinsOS+) |
|---|---|---|
| Гарантія від виробника | Зберігається | Може анулюватись |
| Авто-тюнінг по чипах | Немає | Є |
| Eco-режим | Немає | Є (економія 10–20% електрики) |
| Turbo-режим | Ні | Є (+5–15% хешрейту) |
| Статистика по чипах | Обмежена | Детальна |
| Підходить для | Звичайна експлуатація | Оптимізація під тариф |

Для домашнього та малосерійного майнінгу офіційна прошивка — надійний вибір. Для оптимізації під конкретний тариф кастомна прошивка окупає себе за 1–3 місяці.

## Коли варто прошивати майнер?

1. **Хешрейт нижче специфікацій** — оновлення може виправити баг або додати оптимізацію.
2. **Машина перегрівається або нестабільно працює** — нова версія краще керує тепловим режимом.
3. **Хочете знизити споживання** — Eco-режим кастомної прошивки зменшує витрати на електрику.
4. **Хочете максимальний хешрейт** — Turbo-режим вичавлює з чипів більше, але потребує хорошого охолодження.
5. **Заводська прошивка застаріла** — виробник може закрити підтримку старих версій.

## Як прошити ASIC-майнер: покрокова інструкція

### 1. Завантажте прошивку

- Для Antminer — лише з [oficijnoho sajtu Bitmain](https://www.bitmain.com/support/firmware)
- Для Whatsminer — із [MicroBT](https://www.microbt.com/support)
- Кастомна — з офіційного сайту Vnish або BraiinsOS+

**Важливо:** перевірте сумісність. Прошивка для S21 не підходить для S21 Pro.

### 2. Зайдіть у веб-інтерфейс майнера

Підключіть комп'ютер до тієї ж мережі, що і майнер. Відкрийте браузер і введіть IP-адресу пристрою (зазвичай 192.168.x.x). Логін/пароль за замовчуванням — `root / root`.

### 3. Завантажте файл прошивки

У веб-панелі перейдіть до **System → Upgrade**. Виберіть файл прошивки (.tar.gz для Antminer) і натисніть **Flash**. Процес займає 3–7 хвилин.

### 4. Не вимикайте живлення!

Під час прошивки майнер ні в якому разі не можна відключати від мережі. Перебій живлення під час оновлення може «зацеглити» пристрій (brick). Після завершення — пристрій перезавантажиться автоматично.

### 5. Перевірте результат

Увійдіть в інтерфейс знову. Переконайтеся, що версія прошивки оновилася, хешрейт в нормі, всі хешборди визначаються.

## Що може піти не так?

- **Хибна прошивка** — встановлена версія, несумісна з апаратною ревізією майнера. Рішення: завжди завантажуйте з офіційних сайтів.
- **Brick після відключення живлення** — потребує відновлення через UART або SD-карту в сервісному центрі.
- **Зниження хешрейту після оновлення** — буває при переході на нову версію. Вирішується скиданням налаштувань і повторним налаштуванням пулу.
